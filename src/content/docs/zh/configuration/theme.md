# 主题

## WebUI 主题

WebUI 支持亮色和暗色双主题，基于凌霄剑炉配色：

- **暗色**：墨黑 `#0B0E11` 底，纸白 `#E8E4D8` 文字，青锋 `#5FE0C7` 强调，金箔 `#C9A86A` 装饰
- **亮色**：纸白 `#E8E4D8` 底，墨黑 `#0B0E11` 文字，青锋 `#0B8A74` 强调，金箔 `#9d682d` 装饰

## TUI 主题

TUI 终端界面支持自定义颜色方案。

## 字体

- 正文：Inter + Noto Sans SC
- 代码：JetBrains Mono
