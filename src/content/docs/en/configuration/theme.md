# Theme

## WebUI Theme

Dual light/dark theme based on LingXiao sword-forge palette:

- **Dark**: Ink black `#0B0E11` bg, paper white `#E8E4D8` text, blade cyan `#5FE0C7` accent, gold `#C9A86A` decor
- **Light**: Paper white `#E8E4D8` bg, ink black `#0B0E11` text, blade cyan `#0B8A74` accent, gold `#9d682d` decor

## Fonts

- Body: Inter + Noto Sans SC
- Code: JetBrains Mono
