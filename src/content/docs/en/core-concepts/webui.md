# WebUI Command Center

The WebUI is LingXiao's visual command console.

## Core Panels

### Task Panel
- DAG visualization
- Task state transitions
- Dependency graphs
- Block and error indicators

### Agent Panel
- Worker roles and identities
- Real-time progress and output
- Tool call history
- Runtime status monitoring

### Tool Calls
- Parameters, permissions, and results
- Call chain tracing
- Duration and performance analysis

### Code Changes
- File diff comparison
- Git operation records
- Branch and merge management
